# Autonomous FinOps Agent Environment

This is a Reinforcement Learning environment for FinOps, simulating SaaS seat management and LLM routing.

## Features
- SaaS Seat Pruning Logic
- LLM API Tier Routing
- Interactive Swagger UI (Port 7860)
- Deterministic Task Graders (Easy, Medium, Hard)
